# Skills Static Analysis Engine

## 引擎定位

`engines/skills-static` 用于对 Skills 包、脚本、配置与依赖进行静态安全分析。

重点识别高风险能力、敏感调用、危险依赖、可疑规则配置和潜在越权路径。

## 输入与输出

输入：

- 任务信息 `Task`
- Skills 包路径、仓库地址或文件集合
- 静态分析规则参数

输出：

- `StaticAnalysisResult`

## 建议结构

```text
engines/skills-static/
├─ src/          # 解析器、分析器、结果组装
├─ rules/        # 静态检测规则
└─ tests/        # 规则测试与样本测试
```

## 与平台后端的解耦方式

- 后端只负责创建分析任务和接收结果。
- 本引擎独立维护规则体系、文件解析流程和命中逻辑。
- 风险命中结果通过统一结构回传，平台统一展示，不依赖引擎内部实现。

## 第一阶段建议

- 优先定义规则命中对象 `RuleHit`。
- 先支持少量高价值规则，如命令执行、文件写入、网络访问、敏感依赖检测。
- 在 `samples/skills` 中维护可重复使用的测试样本。
