针对您负责的智能体暴露面调研、特征提取、扫描探针设计以及准确率优化等工作，我为您整理了以下高度相关的前沿学术论文与工业界技术文献，可以为后续工作提供理论和工程参考：



\*\*1. 智能体暴露面调研与 API/端口特征提取\*\*

\*   \*\*Unveiling the Landscape of LLM Deployment in the Wild: An Empirical Study\*\* \[1]

&#x20;   \*   \*\*核心参考价值：\*\* 该研究对真实网络环境中的大模型与智能体部署进行了大规模测绘，发现了超过32万个公开暴露的服务。论文详细梳理了15种主流框架的158个特有 API 路由特征，并分析了默认端口和 HTTP 响应头等指纹信息，是进行暴露面调研的绝佳基线数据 \[2]\[1]。

\*   \*\*Model Context Protocol (MCP) Security Best Practices \& Threat Landscape\*\* \[3]

&#x20;   \*   \*\*核心参考价值：\*\* 随着 MCP 协议成为智能体调用外部工具的标准，MCP 暴露面成为了新的资产测绘重点。相关研究梳理了智能体与外部工具交互过程中的安全特征、API 鉴权方式以及高危攻击面，有助于完善 API 路由和依赖组件的特征提取 \[3]。
https://arxiv.org/html/2505.02502v2
https://arxiv.org/html/2505.02502v1



\*\*2. 扫描探针设计与主动指纹识别\*\*

\*   \*\*LLMmap: Fingerprinting For Large Language Models (USENIX Security 2025)\*\*

&#x20;   \*   \*\*核心参考价值：\*\* 介绍了针对大模型应用的第一代主动指纹识别技术。该方法通过发送精心构造的探测请求并分析响应，仅需约 8 次交互即可准确识别 42 种不同的模型版本，准确率超过 95%。该论文的探针构造逻辑对优化扫描探针设计有极高的参考价值。

\*   \*\*Julius: Open Source LLM Service Fingerprinting Tool\*\* \[4]

&#x20;   \*   \*\*核心参考价值：\*\* Praetorian 发布的开源 LLM 服务指纹识别工具。它采用声明式的 YAML 配置文件作为扫描探针，并引入了基于端口的优先级划分、单飞缓存（Singleflight）以及指纹特异度评分（Specificity Scoring）来消除误报 \[4]\[4]。这对你们设计自动化的“扫描探针与指纹样本表”提供了直接的工程参考。

\*   \*\*Fingerprinting of Large Language Models (arXiv:2501.18712)\*\* \[5]

&#x20;   \*   \*\*核心参考价值：\*\* 该论文提出了一种融合静态与动态指纹识别技术的混合流水线，专门解决真实世界中多智能体系统、频繁更新以及无法获取内部参数情况下的指纹识别难题 \[5]。
https://datadome.co/agent-trust-management/mcp-security/
https://github.com/praetorian-inc/julius



\*\*3. 动态/加密流量侧的指纹识别补充\*\*

\*   \*\*AgentPrint: Exposing LLM User Privacy via Traffic Fingerprint Analysis\*\* \[6]

&#x20;   \*   \*\*核心参考价值：\*\* 当静态端口或 HTTP 头部被 WAF 或 TLS 隐藏时，该研究提供了一种流量侧的识别方法。它将数据包序列转化为多视角流量聚合矩阵（MTAM），并利用 CNN 网络提取智能体的行为指纹（F1-score达0.866） \[6]。这可以作为提升识别结果覆盖面的进阶手段。
https://arxiv.org/html/2501.18712v4
https://arxiv.org/pdf/2510.07176



\*\*4. 资产维度拓展与准确率优化\*\*

\*   \*\*Fingerprinting AI Agent Skills: The Missing Identity Layer for Agent Plugins\*\* \[7]

&#x20;   \*   \*\*核心参考价值：\*\* 针对智能体插件（Skills）的资产化管理，业界提出了对整个插件目录树结构进行哈希计算的内容感知指纹技术（Skill ID） \[7]。这对于完善“指纹样本表”中关于智能体外部依赖资产的测绘非常有帮助。

\*   \*\*Jenius Agent: Towards Experience-Driven Accuracy Optimization in Real-World Scenarios\*\*

&#x20;   \*   \*\*核心参考价值：\*\* 该文献专注于智能体在真实场景中长期执行任务时的稳定性与准确率优化，引入了联合测量程序保真度、语义正确性和执行效率的评估方法，可为“后续扫描测试与准确率优化”环节提供测试指标参考。
https://www.gendigital.com/blog/insights/research/ai-agent-skill-fingerprinting
